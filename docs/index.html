PCD Manager

An elegant application to manage every layer of your PCD lifecycle.

Why PCD Manager Matters

PCD Manager is more than a dashboard — it’s a unified control plane to simplify PCDs lifecycle operations. It enables speed, accuracy, and visibility for your PCD.

✅ Reduces Human Error

Validations, syncs, and stateful tracking prevent misconfiguration.

⏱ Faster Ops

Region onboarding, upgrades, and deletes in a few clicks.

📣 Seamless Alerts

Slack notifications keep teams informed and responsive.

Unlocking the Power of PCD Manager

🔑 OAuth Login

Google OAuth restricted to Platform9 users.

🕵 Auditing

Every interaction with the app will be recorded with user email in logs.

📊 Region Overview

Grouped by customer with task state, lease, tags.

🔍 Search & Filter

Filter by tag, FQDN, owner, environment.

🔄 Reset Task

Set region's task_state back to ready.

🏷️ Tag Management

Add or remove classification tags.

🚀 Create Infra

New Infra region for a customer.

➕ Add Region

Add workload region under PCD.

🔁 Update Lease

Extend lease date, add notes.

❌ Delete Region

Delete workload/Infra + customer.

🔗 Tempus Link

Direct link to upgrade page.

🗑 Auto Cleanup

Automatically deletes expired regions by comparing the current date with the lease.

📣 Slack Alerts

Sends alert messages 7 or 1 day before lease expiry to region owners via Slack.
